Job Name: PR-Verification-Pre-Req

Condition:

This job runs only if the following conditions are met:

  • The state of the PR review is 'APPROVED'.
  • The pull request is not yet merged (github.event.pull_request.merged == false).

Runs on:

api_ct_runner_build

Steps:

  1. Checkout Action: Uses the actions/checkout action to clone the repository at the latest commit.
  2. PR Verification Steps:
    • ID: Action1
    • Run: Executes a script that echoes "Any pre-requisites if any."

This job is designed to execute when a pull request review is approved, and the pull request is not yet merged. It runs on a custom runner named api_ct_runner_build and performs basic steps such as checking out the repository and executing a script related to PR verification steps. You can customize the script inside the run block to include any specific pre-requisites or verification steps you require for your pull requests.
